Java OOP Concepts – Practice Files

📌 Description

This repository contains Java programs demonstrating fundamental Object-Oriented Programming (OOP) concepts. Each file focuses on a specific concept to help understand how OOP works in practice.

---

🧠 Concepts Covered

- Inheritance (Single, Multilevel)
- Classes and Objects
- Method Overriding
- Basic OOP structure

---

📂 Files Included

- File 1: Demonstrates basic class and object creation
- File 2: Example of single inheritance
- File 3: Example of multilevel inheritance
- File 4: Method overriding concept
- File 5: Another inheritance-based example
- File 6: Practice implementation combining concepts

---

▶️ How to Run

1. Open the files in any Java IDE (like IntelliJ or VS Code)
2. Compile the program
3. Run the main class file

---

💡 Purpose

These programs are created for learning and practicing OOP concepts in Java, especially inheritance. They are suitable for beginners and first-year students.

---

🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
